## Kernidee

Cora ist kein kompliziertes Kochbuch.
Cora ist ein Human-First-Alltagssystem, das Familien entlastet.

Ziel:
- weniger Kopfstress
- weniger Einkaufschaos
- Budget schützen
- einfache Kindergerichte
- Low-Energy und No-Sidequests
- Rezepte → Wochenplan → Einkaufsliste

## Leitsatz

Nicht perfekt planen.
Nur den Kopf entlasten.
